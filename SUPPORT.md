# Project IMACE Support

Project IMACE is an open research initiative. Support is provided through structured, community-driven workflows.

---

## 1. How to Get Support

### GitHub Issues (Primary)

Use GitHub Issues for most queries:

- select the appropriate issue template  
- clearly describe your question or problem  
- provide relevant context  

Use for:

- research questions  
- documentation clarification  
- workflow guidance  
- suggestions and discussions  

---

### Direct Contact

For coordination or formal matters:

- research@imace.online  
- coordinator@imace.online  
- cro@imace.online  

---

### Sponsorship

For sponsorship and resource support:

- commons@imace.online  

Refer to:  
- [Sponsorship](https://github.com/project-imace/charter/blob/main/SPONSORSHIP.md)

---

## 2. Before Requesting Support

- review documentation  
- check existing issues  
- ensure your query is clear  

Refer to:

- [Charter](https://github.com/project-imace/charter/blob/main/CHARTER.md)  
- [Research Scope](https://github.com/project-imace/charter/blob/main/RESEARCH_SCOPE.md)  

---

## 3. Expectations

- be clear and concise  
- follow issue templates  
- remain respectful (see [Code of Conduct](CODE_OF_CONDUCT.md))  

---

## 4. Security Issues

Do not report security issues here.  
Refer to [SECURITY.md](SECURITY.md).

---

## Summary

Use GitHub Issues for most support.  
Use email only when necessary.
